# ◈ Sistema de Autenticação e Dashboard

Um sistema completo de login, cadastro e dashboard de usuários, construído com foco em uma interface limpa, moderna e responsiva. O projeto está sendo estruturado para integrar um frontend em Vanilla JS com um backend robusto em Node.js e MongoDB.

## ✨ Funcionalidades

- **Cadastro de Usuários:** Criação de conta com validação de formulário.
- **Login e Autenticação:** Sistema de acesso seguro com opção "Lembrar-me".
- **Dashboard Interativo:** Painel de controle do usuário logado.
- **Gestão de Sessão:** Controle de tempo de sessão e expiração.
- **Log de Acessos:** Tabela com histórico de acessos do usuário (sucesso, falha, logout).
- **Proteção de Rotas:** Redirecionamento automático para usuários não autenticados.

## 🛠️ Tecnologias Utilizadas

### Frontend
- **HTML5 & CSS3:** Semântica e estilização moderna com variáveis CSS.
- **JavaScript (Vanilla):** Lógica de interface, manipulação do DOM e simulação de autenticação (LocalStorage/SessionStorage).

### Backend
- **Node.js:** Ambiente de execução.
- **Express:** Framework para roteamento e criação da API.
- **MongoDB & Mongoose:** Banco de dados NoSQL e modelagem de dados (em integração).
